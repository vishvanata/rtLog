The only real-time log of all unmanned vehicles available for public to use. Developers can use our API to create their own applications to display the data on mobile devices and automobiles. Active vehicles around a 1 mile Radius of where you are can be fetched and displayed on a map plotting the vehicals. Very interesting and attractive apps can be created. Simple to use the REST API.
